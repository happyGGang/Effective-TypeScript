# 5주차 : 타입 추론

## item 19 : 추론 가능한 타입을 사용해 장황한 코드 방지하기

1. 추론 가능한 타입은 타입구문 쓰지말자
   ```ts
   // 이렇게 해도 되지만 추론 가능하다면
   const num: number = 1;
   // 이렇게 쓰자
   const num = 1;
   ```
2. 타입이 변경될수도 있다면 디스트럭쳐링 쓰자.
3. 추론 가능한 경우라도 함수 반환이나 객체 리터럴에는 타입 명시를 고려하자
4. no-inferrable-types 쓰면 타입구문이 필요한지 안한지 확인 가능

## item 20 : 다른 타입에는 다른 변수 사용하기
